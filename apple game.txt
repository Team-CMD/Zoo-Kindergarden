#플레이어 좌표 (x,y)


#플레이어 초기화 (가운데에서 시작)
플레이어 좌표(y) = 화면 높이 - 플레이어의 길이
플레이어 좌표(x) = 화면 가로 길이 /2 - 플레이어 길이 /2

#왼쪽 방향키 -> 왼쪽
if 키보드 왼쪽키 입력:
   goto(x-10, y)

-> 왼쪽 끝으로 갈 경우 이동 못하게
if 플레이어 좌표(x) < 0:
    플레이어 좌표 == 0


#오른쪽 방향키 -> 오른쪽
if 키보드 오른쪽키 입력:
   goto(x+10, y)

-> 오른쪽 끝으로 갈 경우 이동 못하게
if 플레이어 좌표(x) > 화면 크기 x 좌표:
    플레이어 좌표 == 화면 크기 x 좌표
